# 🌤️ Weather App - Previsão do Tempo

Aplicação web que permite consultar a previsão do tempo de qualquer cidade do mundo em tempo real.
O projeto consome dados da API da OpenWeather e exibe informações como temperatura, descrição do clima e ícone representando as condições meteorológicas.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* API de clima da OpenWeather
* Fetch API
* Manipulação do DOM

## 📸 Funcionalidades

* Buscar clima de qualquer cidade
* Exibir temperatura atual
* Mostrar descrição do clima
* Exibir ícone correspondente às condições climáticas
* Interface simples e intuitiva

## 🧠 Conceitos praticados

Este projeto foi desenvolvido para praticar:

## ⚙️ Como executar o projeto

1. Clone o repositório

git clone https://github.com/seu-usuario/weather-app

2. Abra a pasta do projeto

3. Abra o arquivo `index.html` no navegador
   ou utilize a extensão **Live Server** no VS Code.

## 🔑 Configuração da API

Este projeto utiliza a API da OpenWeather.

1. Crie uma conta em:
   https://openweathermap.org/

2. Gere uma API Key

3. No arquivo `script.js` substitua:

const apiKey = "SUA_API_KEY_AQUI"

pela sua chave da API.
